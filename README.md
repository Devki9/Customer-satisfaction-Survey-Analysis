# Customer Satisfaction Survey Analysis
### A Case Study of Invistico Airlines

## Project Overview

Customer satisfaction is a key indicator of an airline's service quality and overall business performance. This project analyzes customer satisfaction survey data from Invistico Airlines to identify the factors that influence passenger satisfaction and provide insights that support data-driven decision-making.

Using Python and data science techniques, the project demonstrates the complete data analysis workflow, including data collection, cleaning, exploratory data analysis (EDA), visualization, and interpretation of results.



## Problem Statement

Invistico Airlines collects customer satisfaction survey data to evaluate passengers' experiences. However, without proper analysis, it is difficult to identify the key factors affecting customer satisfaction and determine which areas require improvement.

This project analyzes customer satisfaction survey data to discover patterns, trends, and relationships between customer characteristics, service quality, and overall satisfaction. The findings can help airline management improve customer experience and enhance service delivery.



## Project Objectives

The objectives of this project are to:

- Load and explore the customer satisfaction dataset using Python.
- Clean and prepare the dataset for analysis.
- Perform exploratory data analysis (EDA).
- Identify factors that influence customer satisfaction.
- Create meaningful data visualizations.
- Draw conclusions and provide recommendations based on the findings.

## Target Users

The analysis is intended to support:

- Airline Management
- Customer Experience Teams
- Operations Managers
- Marketing Department
- Business Analysts


## Dataset

**Dataset Name:** Invistico Airline Customer Satisfaction Dataset

**Source:** Kaggle

The dataset contains customer demographic information, travel details, service ratings, delays, and overall customer satisfaction.
